API Usage

To use the chatbot API in your own application, send a POST request to:

POST https://unani-chatbot.onrender.com/ask Content-Type: application/json { "question": "Your question here" }

The API returns a streaming response with the chatbot's answer.

JavaScript Example

async function askChatbot(question) { const response = await fetch('https://unani-chatbot.onrender.com/ask', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ question }), }); if (!response.ok) { throw new Error('HTTP error! status: ' + response.status); } const reader = response.body.getReader(); const decoder = new TextDecoder(); let fullResponse = ''; while (true) { const { value, done } = await reader.read(); if (done) break; const chunk = decoder.decode(value, { stream: true }); fullResponse += chunk; // Process chunk as it arrives for real-time updates console.log(chunk); } return fullResponse; }
